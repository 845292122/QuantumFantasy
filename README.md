# Quantum Fantasy

## 项目简介

基于react+express的后台管理模板,使用pnpm + monorepo构建

## 技术栈

### 前端

- react18
- ant-design
- vite
- jotai
- ahooks

### 后端

- express
- prisma
- mysql

## 相关技术文档

- 样式方案: [antd-style](https://ant-design.github.io/antd-style/zh-CN)
- 图标方案: [icon-park](https://iconpark.oceanengine.com/official)
